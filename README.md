## 📖 Project Overview

OptiCrop is an AI-powered Smart Agricultural Production Optimization Engine developed to help farmers select the most suitable crop based on soil nutrients and environmental conditions.

The application uses a **Random Forest Machine Learning model** trained on agricultural datasets to predict the most appropriate crop using:

- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- Soil pH
- Rainfall

# 🚀 Installation

Clone Repository

```bash
git clone https://github.com/lokii2109/OptiCrop-Smart-Agricultural-Production-Optimization-Engine.git
```

Move to Project

```bash
cd OptiCrop-Smart-Agricultural-Production-Optimization-Engine
```

Install Dependencies

```bash
pip install -r requirements.txt
```

Run Application

```bash
python app.py
```
